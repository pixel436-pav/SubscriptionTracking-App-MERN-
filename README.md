# SubscriptionTracking-App-MERN-
A RestFul API to track Subscriptions of the user, also reminding them when to pay it using by emailing them

---

## Features
- Create Update Delete Subscriptions (CRUD)
- Track Biling Cycles 
- Automatic Renewal Date

## Tech Stack 
- Backend: ExpressJs 
- frontend : ReactJs
- Databse: MongoDB
- API Style: REST
